# sql-job-analysis
This project analyzes a job portal database using SQL.  
It demonstrates database design, data insertion, and analytical queries.

## Database Tables
- candidates
- companies
- jobs
- applications

## SQL Concepts Used
- Joins
- Group By & Having
- Subqueries
- Case When
- Aggregate functions

## Tools Used
- MySQL

> All queries were executed and tested using MySQL Workbench.
